# System Optimizer & Search Utility

Инструмент автоматизации для сканирования дискового пространства, быстрого поиска файлов и безопасной очистки системы от временных файлов (мусора). Программа оснащена графическим интерфейсом на базе библиотеки `Tkinter` и разработана с использованием классических динамических структур данных.

---

##  Основные возможности

*   **Сканирование и поиск мусора:** Быстрый поиск системного мусора по заданным расширениям (`.tmp`, `.log`, `.bak`, `.old`).
*   **Гибкий поиск файлов:** Встроенная функция поиска конкретного файла по части его имени в выбранной директории.
*   **Массовое управление результатами:** Возможность выделять файлы пачками (через `Ctrl` / `Shift`) или выбрать всё в один клик для последующего удаления.
*   **Безопасная корзина (Карантин):** Удаляемые файлы не стираются навсегда, а перемещаются в изолированную локальную директорию `_quarantine`.
*   **Пакетный откат изменений (Undo):** Возможность мгновенно восстановить последнюю удаленную группу файлов в их исходные папки.
*   **Многопоточность (Multithreading):** Сканирование файловой системы выполняется в отдельном потоке, благодаря чему интерфейс программы не зависает во время работы.

---

##  Реализованные структуры данных

Вся логика управления данными написана вручную без использования встроенных контейнеров Python (таких как `list` или `collections.deque`), что демонстрирует низкоуровневую работу с памятью и указателями:

1.  **Очередь :** Используется для обхода файловой системы (алгоритм BFS / поиск в ширину) при сканировании директорий. Обеспечивает последовательную и стабильную обработку путей.
2.  **Стек :** Управляет историей удалений. При удалении файлы группируются в пакеты и помещаются в стек. Кнопка «Восстановить» извлекает последний пакет по принципу LIFO (последним пришел — первым ушел).
3.  **Дек :** Двусвязный список, отвечающий за «Журнал действий» (статус-бар). Он жестко ограничен 5 элементами: при добавлении нового лога справа, старый лог автоматически выталкивается слева.
