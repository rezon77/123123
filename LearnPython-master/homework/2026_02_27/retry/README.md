# Домашнее задание

Сделайте декоратор, который будет повторно вызвать функцию, если при выполнении она вызвала исключение.

* Если функция вызвала исключение ValueError, то необходимо просто повторно вызвать функцию 
* Если функция вызвала исключение OsError, то необходимо написать в stdout: 
  ```  
  "{func.__name__} raise OsError exception."
  ```
  

Функция, которая будет задекорирована уже написана в файле main.py.
